# q(exit this session)

## 概述

退出crash会话，等同于quit

```shell
crash> alias
ORIGIN   ALIAS    COMMAND
builtin  man      help 
builtin  ?        help 
builtin  quit     q 
```


## 帮助信息

* <https://crash-utility.github.io/help_pages/q.html>

```
NAME
  q - exit this session

SYNOPSIS
  q  

DESCRIPTION
  Bail out of the current crash session.

NOTE
  This command is equivalent to the "exit" command.
```
